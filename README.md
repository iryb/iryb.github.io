# Tasks dashboard

This is simple React app for task management.

## Demo

[View demo](https://tasks-dashboard-49705.firebaseapp.com/)

## Features

- Authentication
- Task creation and editing
- Search
- Task assignments
- Comments
- Personal dashboard

## Tech stack

- React
- Redux Toolkit
- Bootstrap
- Firebase
- HTML
- SCSS
